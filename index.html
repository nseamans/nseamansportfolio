Event: Online Task Scheduling and Grouping Application

 
Event View Tab
Dashboard Tab

About this Project

Event allows users to view and edit events organized by groups. The application also contains a dashboard that displays statistics on the events with a pie-chart included. The database is hosted over Mongo Atlas and the application over Heroku. I have also built in several queries which will allow users to search for events based on event type or group type.

Technologies and Libraries:

  • Front-end: HTML 5, CSS 3, JQuery, BootStrap 4
  • Back-end: Node.js, Express, EJS, MongoDB
  • Hosting: Heroku, Mongo Atlas

Features

  • User Authentication
  • Users can Edit existing
  • Responsive and Material Design
  • Ability to search query information on events and groups
  • Interface that utilizes color cordination to organize data

Major/Minor Chord Application

 
 

About this Project

An application that allows a user to look up major and minor triads. Triads that have keys above the keys an octave above the original are stressed with a +1 below to indicate the fact. A recommendation is then provided based on the third note of the triad.

Technologies and Libraries:

  • Front-end: HTML 5, CSS 3, JavaScript, JQuery

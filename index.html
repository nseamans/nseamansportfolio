About this Project

Event allows users to view and edit events orgainized by groups. The application also contains a dashboard that displays statistics on the events with a pie-chart included. The database is hosted over Mongo Atlas and the application over Heroku. I have also built in several queries which will allow users to search for events based on event type or group type.

Technologies and Libraries:

  • Front-end: HTML 5, CSS 3, JQuery, BootStrap 4
  • Back-end: Node.js, Express, EJS, MongoDB
  • Hosting: Heroku, Mongo Atlas

Features

  • User Authentication
  • Users can Edit existing
  • Responsive and Material Design
  • Ability to search query information on events and groups
  • Interface that utilizes color cordination to organize data

BitCoin Info Application

 
 

About this Project

This application functions as a static dashboard for some of the top bitcoin performers. All data is pulled from an open API. Sections include:

  • Head Line Article. The most recent story headlining on crypto currencies supplies a link to the website the article is pulled from. ( Note: The api used for this application no longer offers this. )
  • Five Well-Known Crypto-Currencies. These where the five best performing Crypto-Currencies as of the time of upload.
  • Daily Market Ratings. The market ratings for 5 of the most notable currencies at the time of the projects creation. Shows the current value and the calculated 24 hour average.
